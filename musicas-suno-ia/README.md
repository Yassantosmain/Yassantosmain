# 🎯 SM8 - Criação de Músicas com Suno IA

##  Descrição
Criação de músicas com Inteligência Artificial.

##  Objetivo
Explorar as capacidades da IA generativa aplicada à música, utilizando a ferramenta Suno IA para criar composições originais a partir de prompts e instruções.

##  Atividades Realizadas
- Exploração da ferramenta Suno IA
- Criação de prompts para geração de músicas
- Testes com diferentes estilos e gêneros musicais
- Análise dos resultados gerados pela IA
