# 🎯 SM5 - Do Clone ao MVP+

##  Descrição
Desenvolvimento de um produto digital, iniciando com um clone e avançando até um MVP viável.

##  Objetivo
Compreender o processo de desenvolvimento de produtos digitais, partindo da reprodução de um sistema existente (clone) e evoluindo até a criação de um produto mínimo viável com funcionalidades próprias.

##  Atividades Realizadas
- Análise e reprodução de um produto digital existente
- Identificação de melhorias e funcionalidades próprias
- Desenvolvimento incremental até o MVP
- Testes e validação do produto final
