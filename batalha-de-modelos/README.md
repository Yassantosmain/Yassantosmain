# 🎯 SM3 - Batalha de Modelos & Engenharia de Prompt (XML)

##  Descrição
Análise comparativa entre modelos de IA e aplicação avançada de XML na organização de prompts.

##  Objetivo
Comparar o desempenho de diferentes modelos de IA para uma mesma tarefa e explorar o uso de XML como estrutura avançada na organização e otimização de prompts.

##  Atividades Realizadas
- Testes comparativos entre diferentes modelos de linguagem
- Análise dos resultados obtidos por cada modelo
- Aplicação de estruturas XML na construção de prompts
- Documentação das diferenças de desempenho entre os modelos
