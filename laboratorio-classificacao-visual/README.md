# 🎯 SM2 - Laboratório de Classificação Visual

##  Descrição
Utilização do Teachable Machine para treinar um modelo de aprendizado de máquina.

##  Objetivo
Explorar o funcionamento do aprendizado de máquina por meio do treinamento de um modelo de classificação visual utilizando a ferramenta Teachable Machine do Google.

##  Atividades Realizadas
- Coleta e organização de imagens para treinamento
- Treinamento do modelo de classificação visual
- Testes e validação dos resultados obtidos
- Análise da performance do modelo treinado
